# Bank Transaction Dashboard (SQL + Power BI)

This project is an end-to-end banking analytics dashboard built using SQL (MySQL) and Power BI.  
It demonstrates how to create a relational database for customers and transactions, extract KPIs such as deposits, withdrawals, monthly trends, and visualize data interactively.

## Project Overview

- **Tech Used:** MySQL, Power BI Desktop, Power Query, Excel
- **Components:**
  - SQL schema for customer & transaction tables
  - KPI/exploratory SQL queries
  - Power BI dashboard for data visualization

## How to Use

1. Run the SQL scripts in MySQL to set up sample data.
2. Connect Power BI to your MySQL instance (`localhost:3306`).
3. Import tables and transform using Power Query.
4. Use provided DAX measures and templates to create the dashboard visuals.

## Features

- Deposit/Withdrawal summary
- Monthly transaction trends
- Customer balance tracking
- Interactive filters and drill-downs

## Files Included

- `Bank Transaction.sql` (SQL file)
- `Bank Transaction Dashboard.pbix` (Power BI dashboard file)
